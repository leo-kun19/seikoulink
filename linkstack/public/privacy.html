Kebijakan Privasi

Terakhir diperbarui: 17 Mei 2026

1. Data yang Kami Kumpulkan

Saat mendaftar dan menggunakan SeikouLink, kami mengumpulkan:

  • Email dan username (untuk autentikasi)
  • Informasi profil (nama tampilan, bio, avatar) yang kamu isi sendiri
  • Link yang kamu tambahkan
  • Data analytics: jumlah kunjungan, klik, dan negara pengunjung (IP di-hash, tidak disimpan mentah)

2. Penggunaan Data

Data digunakan untuk:

  • Menyediakan layanan link-in-bio
  • Menampilkan statistik kunjungan di dashboard
  • Mengirim email verifikasi dan reset password
  • Moderasi konten (mencegah link berbahaya)

3. Penyimpanan Data

Data disimpan di server kami yang berlokasi di Asia. Password di-enkripsi menggunakan bcrypt. IP pengunjung di-hash (tidak bisa di-reverse).

4. Berbagi Data

Kami tidak menjual atau membagikan data pribadi kamu ke pihak ketiga. Kami menggunakan Google Safe Browsing API untuk memeriksa keamanan link (hanya URL yang dikirim, bukan data pribadi).

5. Cookie

Kami menggunakan cookie untuk autentikasi (session login) dan proteksi CSRF. Tidak ada cookie tracking pihak ketiga.

6. Hak Pengguna

Kamu bisa:

  • Mengubah atau menghapus data profil kapan saja dari dashboard
  • Meminta penghapusan akun dengan menghubungi kami

7. Kontak

Pertanyaan tentang privasi bisa dikirim ke: seikou.jokki@gmail.com
